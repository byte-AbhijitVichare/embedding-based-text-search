# Embedding-Based Text Search Engine

This project implements a semantic text search engine using
Sentence-BERT embeddings and cosine similarity.

## Features
- Semantic sentence search
- Transformer-based embeddings
- Streamlit web interface

## Tech Stack
- Python
- Sentence-Transformers
- Scikit-learn
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
